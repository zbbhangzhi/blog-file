/**
 * http://content.research.neustar.biz/blog/hll.html
 * 用于海量数据下不精确统计不重复的个数（基数）
 * 概念：
 * 1.基数：一个集合中不重复元素的个数
 * 2.基数估计法：LogLog算法
 *      1.分桶hash：根据得到的二进制表示的前几位指向属于哪个桶
 *      2.前导零：去除hash值的第一位，连续0的个数加一
 *      3.基数估计 = const * 桶数 * 2 ^ 前导零平均值
 *
 * 3.HyperLogLog
 *      1.改进：平均值改为调和平均值，因为前导零连续为偶然性
 *      2.基数估计 = const * 桶数 * 所有桶估计值的调和平均值
 *      3.数据较少时：基数估计 = 桶数 * log(桶数 / 桶中估计值为0的桶数）
 *      4.统计学上用来衡量估计精度的一个指标是“相对标准误差”(relative standard deviation，简称RSD),RSD的值其实就是（(每次估计的值）在（估计均值）上下的波动）占（估计均值）的比例
 */
